### 目录结构
```
course-name/
├── README.md                   # 课程总览、学习计划
├── SUMMARY.md                  # 目录索引
├── notes/                      # 课程笔记
│   ├── chapter-01/             # 按章节分类
│   │   ├── 01-topic-name.md    # 具体主题笔记
│   │   └─── 02-topic-name.md
│   └── chapter-02/
├── reviews/                    # 阶段性总结
│   ├── week-review.md          # 周复习
│   └── chapter-review.md       # 章节复习
├── projects/                   # 项目实践
├── resources/                  # 课程资源
│   ├── images/                 # 图片资源
│   └── codes/                  # 代码示例
└── summary/                    # 知识总结
    ├── mindmap.md              # 思维导图
    └── key-points.md           # 重点概念
```
